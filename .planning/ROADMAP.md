# Roadmap: yclaude

## Overview

yclaude delivers a local-first analytics dashboard for Claude Code usage, distributed via `npx yclaude`. v1.0 shipped the foundation (parser, cost engine, privacy, basic dashboard). v1.1 completed the full analytics experience and published to npm. v2.0 will add cloud sync and team features.

## Milestones

- ✅ **v1.0 Local MVP** — Phases 1–4 (shipped 2026-02-28)
- ✅ **v1.1 Analytics Completion + Distribution** — Phases 5–10 (shipped 2026-03-05)
- 📋 **v2.0 Cloud & Teams** — Phases 11+ (planned)
