**parser.ts** -- Core parsing logic:

`parseCursorData(options: { globalDbPath: string; workspaces: Array<{ dbPath: string; workspacePath?: string }>; preserveContent?: boolean; debug?: boolean }): UnifiedEvent[]`

Implementation steps:
1. Open global DB via `openCursorDb(globalDbPath)`.
2. Detect schema version via `detectSchemaVersion()`. Log version in debug mode.
3. For each workspace: open workspace DB, read composer heads from `composer.composerData` key in ItemTable via `readKvEntry()`. Parse JSON, validate each head with `safeParseComposerHead()`. Enrich each head with workspacePath from the workspace entry.
4. For each composer head:
   a. Read full composer data from global DB: `readKvEntry(globalDb, 'cursorDiskKV', 'composerData:' + composerId)`. Parse and validate with `safeParseComposerData()`.
   b. Read bubbles from global DB: `readKvEntries(globalDb, 'cursorDiskKV', 'bubbleId:' + composerId + ':')`. Parse each value as JSON, validate with `safeParseBubble()`. Skip (with debug log) any that fail validation.
   c. Filter out user bubbles (type === 1) for event creation -- only AI responses (type === 2) become UnifiedEvent entries. But use user bubbles for timestamp calculations.
   d. If zero valid bubbles after filtering, skip this composer (per decision: "Empty composers excluded").
   e. Extract cost data from composerFullData.usageData: iterate entries, sum costInCents. If total > 0, set costUsd = total / 100, costSource = 'reported'. If no cost data (empty usageData or all zero), set costUsd = 0, costSource = 'reported'.
   f. Determine isAgentic: true if unifiedMode === 'agent', false otherwise. Use composer head's unifiedMode first, fall back to composerFullData's unifiedMode.
   g. Determine model: prefer usageData keys that aren't 'default' (actual model names like 'claude-3.5-sonnet'). Fall back to modelConfig.modelName.
   h. Determine session type: 'edit' if unifiedMode === 'edit', 'composer' otherwise.
   i. Compute session durationMs: sort all bubbles (user + AI) by timestamp, duration = last - first in ms. Set on the FIRST event only (matches Claude pattern where durationMs is per-session).
   j. Create one UnifiedEvent per AI bubble (type === 2):
      - id: crypto.randomUUID()
      - provider: 'cursor'
      - sessionId: composerId
      - timestamp: getBubbleTimestamp(bubble) -- try createdAt (v3 ISO), then timingInfo.clientRpcSendTime (v2 epoch ms -> ISO), then composerHead.createdAt as fallback
      - type: 'assistant'
      - costUsd: divided evenly across AI bubbles in this composer (totalCostUsd / aiBubbleCount), or 0
      - costSource: 'reported'
      - model: from step (g) -- use conditional spread
      - tokens: from bubble.tokenCount if present (map inputTokens->input, outputTokens->output, zeros for cache fields) -- use conditional spread
      - cwd: composerHead.workspacePath -- use conditional spread
      - gitBranch: composerHead.createdOnBranch -- use conditional spread
      - durationMs: on first AI bubble only -- use conditional spread
      - isAgentic: from step (f) -- use conditional spread
      - costInCents: raw total cents from usageData (on first bubble only for session-level data) -- use conditional spread
      - message: if preserveContent, include { text: bubble.text, richText: bubble.richText, thinking: bubble.thinking, images: bubble.images } -- use conditional spread. Only include fields that are present.
5. Close all databases in finally blocks.
6. Return flattened array of all UnifiedEvent[] across all composers.

Internal helpers (not exported):
- `getBubbleTimestamp(bubble: RawBubble, fallbackEpochMs?: number): string` -- Extracts timestamp as ISO string. Priority: bubble.createdAt (v3), timingInfo.clientRpcSendTime (v2, convert epoch ms to ISO), fallbackEpochMs (convert to ISO). Returns current time ISO as last resort.
- `extractCostFromComposer(composerData: ComposerFullData | null): { costInCents: number; models: string[] } | null` -- Extracts and sums costInCents from usageData. Returns null if empty/zero.
- `deriveModel(composerData: ComposerFullData | null): string | undefined` -- Extracts model name from usageData keys (prefer non-'default') or modelConfig.modelName.

Use conditional spread pattern for ALL optional UnifiedEvent fields per project convention (exactOptionalPropertyTypes).
Use debugLog() for verbose logging (bubble counts, parse errors, cost data found).
