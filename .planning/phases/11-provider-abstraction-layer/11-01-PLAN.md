    **Step 1: Create `src/providers/claude/adapter.ts`** -- the ClaudeAdapter class.

    The adapter wraps the existing parse -> privacy-filter -> cost pipeline:

    ```
    class ClaudeAdapter implements ProviderAdapter {
      readonly id = 'claude' as const;
      readonly name = 'Claude Code';

      async detect(): Promise<boolean> -- call discoverJSONLFiles() with no args, return files.length > 0
      async load(opts: LoadOptions): Promise<UnifiedEvent[]> -- full pipeline:
        1. discoverJSONLFiles(opts.dir)
        2. Stream + normalize via DedupAccumulator (same as existing parseAll)
        3. If NOT preserveContent: applyPrivacyFilter
        4. computeCosts on filtered events
        5. Map CostEvent[] to UnifiedEvent[] with field mapping:
           - uuid -> id
           - add provider: 'claude'
           - costUsd: strip EstimatedCost brand (cast to number)
           - add costSource: 'estimated'
           - Preserve all optional fields using conditional spread for exactOptionalPropertyTypes safety
           - Explicitly transfer `message` field from passthrough data when preserveContent=true
        6. Enable debug logging if opts.debug (call enableDebug())
    }
    ```

    Import from the MOVED file locations (e.g., `'./parser/reader.js'`, `'./cost/engine.js'`, etc.)
    Import `enableDebug` from `'../../shared/debug.js'`

    const raw = e as Record<string, unknown>;

    import all three adapters
    const KNOWN_ADAPTERS: ProviderAdapter[] = [new ClaudeAdapter(), new CursorAdapter(), new OpenCodeAdapter()]

    export async function loadProviders(
      opts: LoadOptions & { exclude?: string[] }
    ): Promise<{ events: UnifiedEvent[]; providers: ProviderInfo[] }>

    Implementation per RESEARCH Pattern 2:
