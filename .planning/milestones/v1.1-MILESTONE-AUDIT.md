### Integration Gap

**INT-01: /api/v1/chats/:id missing subagent cost split fields** (Low severity)

- `ChatSummary` interface in `useChatDetail.ts:33-35` declares `mainCostUsd`, `subagentCostUsd`, `hasSubagents` as required
- API route at `api.ts:1046` does not return these fields
- Runtime behavior: `hasSubagents` is `undefined` (falsy) → conditional safely suppresses the split. No crash.
- Impact: Subagent cost breakdown never appears in ChatDetail even for multi-agent sessions
- Fix: 3-line addition to chats/:id handler, mirroring sessions/:id pattern (api.ts:795-799)
- Affected: CHAT-01 (core chat viewing works; only cost split presentation gap)
